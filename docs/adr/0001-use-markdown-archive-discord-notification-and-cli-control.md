# Use Markdown Archive, Discord Notification, and CLI Control

The complete Daily Brief is stored in a Markdown-first Brief Archive, while Discord Delivery is limited to automatic notification or summary push and the MVP Operational CLI owns explicit workflow control, source management, regeneration, and status inspection. This keeps long-term knowledge, delivery notification, and operational control separate instead of turning Discord into the source of truth or a control surface; the richer chat-based Control TUI is deferred to V2.
